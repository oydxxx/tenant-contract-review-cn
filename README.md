# 中国大陆住宅租赁合同审查与谈判 Skill

这是一个面向中国大陆住宅租客的开源 Codex Skill。目标是在签约前帮助租客识别合同风险、补齐关键约定、准备可执行的修改文本和谈判方案，并在证据允许的范围内形成签约建议。

> 当前状态：U1 可安装安全骨架。宿主门禁、材料状态、法源、决策、谈判和终稿复核能力尚未完成发布验证。请勿用当前版本处理真实合同或作出真实签约决定。

## 面向谁

- 正在签署中国大陆住宅租赁合同的租客；
- 需要把合同问题转化成具体修改文字和沟通方案的人；
- 希望贡献官方城市规则或可复现风险场景的维护者。

本项目不覆盖港澳台、境外租赁、商铺或办公室租赁，也不代理诉讼、仲裁或向第三方发送法律函件。

## 四档结论

完整版本只会输出以下四档总体结论之一：

1. “基于已提交并核实的材料，未发现需要阻止签约的重大风险”；
2. “修改后再签”；
3. “暂停签约并核实”；
4. “建议放弃”。

第一档不是“合同安全”保证；第四档只允许在材料完整、关键事实已核实、高置信度命中预定义严重红线时使用。材料不完整、OCR 关键字段未确认或地方规则未核实时必须降级，不能用确定语气掩盖未知项。

## 条件能力与首个参考宿主

首个参考宿主是 OpenAI Codex 的 Skill 运行方式。本仓库只验证 Skill 的安装与发现，不把平台能力推定为本项目已经取得的机器可验证证明。

| 运行配置 | 文件安全声明 | 隐私与案件隔离声明 | 当前允许输入 |
| --- | --- | --- | --- |
| OpenAI Codex + 本 U1 包 | 未由本包验证，视为 `unknown` | 未由本包验证，视为 `unknown` | 仅充分去标识化的粘贴文本；不读取附件 |
| 其他未集成宿主 | `unknown` | `unknown` | 仅充分去标识化的粘贴文本；不读取附件 |

这里的 `unknown` 只表示本仓库尚无可信能力声明，不表示对宿主平台安全能力作负面判断。后续只有文件真实类型、危险能力禁用、资源限制和隔离解析均验证通过时才可读取附件；只有加密、案件访问控制、日志排除、保留删除和第三方流转边界也验证通过时才可读取未经脱敏的真实合同。

## 安装

取得本仓库后，在仓库根目录执行：

```sh
mkdir -p "${CODEX_HOME:-$HOME/.codex}/skills"
cp -R skills/tenant-contract-review-cn "${CODEX_HOME:-$HOME/.codex}/skills/"
```

重新启动或刷新宿主后，应能发现 `$tenant-contract-review-cn`。安装时只复制 `skills/tenant-contract-review-cn/`，不要复制根目录的贡献、发布或测试文件。

## 最短示例

先只验证 Skill 是否可发现，不提交真实合同：

```text
使用 $tenant-contract-review-cn。先说明当前宿主的材料准入边界；不要读取附件，也不要给出最终签约结论。
```

完整能力发布后，典型请求会是：“请审查这份签约前住宅租赁合同，先检查宿主能力和材料完整性，再列出风险、建议条款和谈判方案。”

## 验证

```sh
python3 "$SKILL_CREATOR_DIR/scripts/quick_validate.py" skills/tenant-contract-review-cn
python3 -m unittest tests.test_publication_safety
```

`SKILL_CREATOR_DIR` 应指向本机安装的 `skill-creator` 目录，该位置不应写入仓库配置。自动检查只证明已编码的结构与门槛成立，不证明法律内容正确，也不替代人工复核。
