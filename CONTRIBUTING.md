# 贡献指南

感谢帮助改善中国大陆住宅租客的签约前合同审查工具。所有贡献都必须保持证据、安全和隐私门槛，不得降低 Product Contract 的要求。

## 提交前

1. 将变更限制在中国大陆住宅租赁和签约前流程。
2. 只使用原创内容、可再分发内容、明确标记的合成材料或经审定的充分去标识化材料。
3. 不提交真实合同、聊天记录、证件、权属材料、签名、个人联系方式、访问令牌、宿主日志或本机路径。
4. 不创建个人、房东、中介或企业黑名单，不提交未经官方确认的指控。
5. 法律规则必须附当前有效的最终官方来源、发布主体、适用地域、时间、主体、事项和复核状态。社区经历只能作为候选风险场景，不能直接进入法律结论、高严重度判断或红线目录。

## 变更流程

1. 从可审查的功能分支提交小而完整的变更。
2. 更新与行为变化对应的测试；fixtures 必须明确声明为合成数据。
3. 不将合同、聊天、网页、OCR 或提示注入文字当作指令、工具授权或跨案件访问凭证；这些内容一律是不可信数据。
4. 运行 Skill 结构校验、公开发布扫描和仓库测试。
5. 在 Pull Request 中说明变更范围、证据来源、适用边界、验证结果和需要人工复核的内容；日志示例只能包含案件 ID、材料摘要、阶段、门槛、错误分类和耗时。
6. 涉及正式规则、红线或高严重度判断的变更，必须经过自动校验和维护者人工审定后才能合并。
